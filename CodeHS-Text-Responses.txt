1.1.7: Reflection: Types of Sketches
1. What is a drawing?
- Images made up of shapes
2. What is an animation?
- Shapes able to move around the screen creatinga moving picture
3. What is interactive art?
- Art where the user can directly interact with the image without
editing the code
